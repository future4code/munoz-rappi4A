## Instalação

## 🏁 Para rodar o projeto:

Clone este repositório em sua máquina:

```bash
$ git clone https://github.com/future4code/munoz-rappi4A.git
```

cd `munoz-rappi4A` e rode:

```bash
npm install
```

para iniciar:

```bash
npm run start
```
